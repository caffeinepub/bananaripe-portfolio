# Personal Portfolio Website

## Overview
A modern, premium personal portfolio website showcasing professional work and experience with a clean, minimal aesthetic. The application is deployed to live production and publicly accessible.

## Core Features

### Design System
- Premium typography using Playfair Display for headings and Inter for body text
- Minimal, modern aesthetic with glassmorphism effects and soft shadows
- Wide spacing and clean grid layouts
- Dark/light mode toggle functionality
- Fully responsive design across all devices
- Accessible color contrast and ARIA labels

### Animations
- Page-load animations for hero section
- Scroll-based reveal animations for all sections
- Hover effects on interactive elements (buttons, cards, links)
- Smooth, unobtrusive transitions throughout

### Website Sections

#### Hero Section
- Professional name and role
- Short value statement
- Two call-to-action buttons: "View Projects" and "Contact Me"

#### About Section
- Professional photo
- Short biographical description
- Key skills displayed as badges or icons

#### Projects Section
- Project showcase cards with images
- Project titles and technology stack information
- Hover effects revealing project links

#### Experience/Services Section
- Professional roles or technical services offered
- Animated card layout

#### Contact Section
- Social media icons
- Email call-to-action (display only, no email sending functionality)

#### Footer
- Minimal branding
- Copyright information

## Technical Requirements

### Technology Stack
- React with Vite
- Tailwind CSS for styling
- Framer Motion for animations
- Lucide React for icons
- Google Fonts integration

### Code Structure
- Modular, reusable React components
- Semantic HTML with proper ARIA roles
- Clean folder organization: components/, sections/, assets/
- Performance optimizations including lazy loading
- Realistic professional sample content

### Accessibility
- Proper semantic HTML structure
- ARIA labels and roles
- Keyboard navigation support
- Screen reader compatibility
- Color contrast compliance

## Content Requirements
- All content in English
- Professional, realistic sample content
- No placeholder text (lorem ipsum)
- Industry-appropriate project examples
- Professional biographical information

## Deployment
- Application deployed to live production environment
- Publicly accessible via web URL
- All generated assets integrated and functional
- All features maintained in production (responsive layout, dark/light mode, animations)
